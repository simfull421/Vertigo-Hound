1인칭 파쿠르: V자 진자운동(V-Pendulum) 수학적 정의서이 문서는 1인칭 카메라의 역동적인 움직임을 제어하기 위한 수리적 모델입니다. 모든 변위는 하나의 공통된 위상(Phase)을 공유합니다.1. 핵심 계수 (Multipliers)모든 최종 계산값에는 다음 두 가지의 상태 계수가 곱해집니다.상태 계수 ($M_{state}$): * 달리기(Sprint)일 때: $1.0$걷기(Walk)일 때: $0.5$ (달리기의 정확히 절반 강도)노력/가속 계수 ($M_{effort}$):도움닫기 및 가속 중 (현재 속도 < 최고 속도의 90%): $1.0$ (100% 쿵쿵거림)최고 속도 크루징 (현재 속도 $\ge$ 최고 속도의 90%): $0.5$ (50% 안정화됨)※ $M_{effort}$는 상태가 변할 때 선형 보간(Lerp)으로 부드럽게 전환되어야 함.2. 기저 위상 함수 (Base Phase Function)시간의 흐름에 따라 $-1$ 에서 $1$ 사이를 왕복하는 사인(Sine) 파동입니다.$$P(t) = \sin(t \times \text{Frequency})$$$P = 0$ : 중앙 (공중에 떠 있는 상태)$P = 1$ : 오른쪽 끝 (오른발 착지)$P = -1$ : 왼쪽 끝 (왼발 착지)3. 4대 궤적 공식 (Trajectory Formulas)위상 $P(t)$를 바탕으로 카메라의 위치와 회전을 결정하는 4가지 절대 공식입니다. (여기서 $A$는 각 변위의 최대 진폭/각도를 의미합니다.)① 수평 진자 이동 (X축 변위)공식: $X = P(t) \times A_x$역학: 위상과 동일하게 왼쪽(-1)과 오른쪽(1)으로 이동.② V자 강하 (Y축 변위)공식: $Y = -|P(t)| \times A_y$역학: 절댓값을 씌우고 마이너스를 붙임. 중앙($0$)에서는 가장 높고, 양 끝($1, -1$)에 도달할 때마다 아래로 뾰족하게 푹 꺼지는 완벽한 v v v 궤적 생성.③ 반대 틸트 (Z축 회전 / Roll)공식: $Roll = -P(t) \times A_{roll}$역학: 위상에 마이너스를 곱s함. 카메라가 왼쪽($-1$)으로 가면 고개는 오른쪽($+1$)으로 꺾이고, 오른쪽($1$)으로 가면 왼쪽($-1$)으로 꺾이는 원심력/관성 구현.④ 앞숙임 (X축 회전 / Pitch)공식: $Pitch = |P(t)| \times A_{pitch}$역학: 절댓값을 씌움. 중앙($0$)에서는 허리를 펴고, 양쪽 끝($1, -1$ / 발이 땅에 닿을 때)에서 고개가 앞으로 끄덕여짐.4. 최종 출력 공식 (Final Output)AI는 매 프레임 위 4가지 공식에 두 가지 계수를 곱하여 최종 오프셋을 산출해야 합니다.$\text{Final } X = ( P(t) \times A_x ) \times M_{state} \times M_{effort}$$\text{Final } Y = ( -|P(t)| \times A_y ) \times M_{state} \times M_{effort}$$\text{Final } Roll = ( -P(t) \times A_{roll} ) \times M_{state} \times M_{effort}$$\text{Final } Pitch = ( |P(t)| \times A_{pitch} ) \times M_{state} \times M_{effort}$
